# Página de cumpleaños para Pao - versión carrusel

Esta versión está pensada para que las fotos NO se corten.
Todas usan `object-fit: contain`, por eso se ven completas dentro de cada tarjeta.

## Cómo usarla

Coloca tu carpeta `imagenes` junto a estos archivos:

```txt
cumple-pao-html-carrusel/
  index.html
  styles.css
  script.js
  imagenes/
    boda/
    bloppers/
    losDos/
    pao/
```

Abre `index.html` con doble clic.

## Importante

Tu carpeta de bloopers está escrita como `bloppers`, por eso el código usa:

```txt
imagenes/bloppers/
```

Si cambias el nombre a `bloopers`, también cambia la ruta en `script.js`.

## Música opcional

Si quieres música, coloca un archivo llamado:

```txt
cancion.mp3
```

junto a `index.html`.
